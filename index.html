A propos

Issue d'un bac ES, je me dirige vers le design depuis que je suis en terminal où j'ai découvert cette sensibilité artistique. Cet autre support du design est, pour ma part, une connaissance importante à acquérir aujourd'hui dans les métiers du Design. J'ai alors réalisé cette page web en code html et css à l'aide du logiciel Atom.
